ToDoTasks

Organisez votre vie, simplement.

À propos

ToDoTasks est une application native macOS conçue pour la simplicité et l'efficacité. Elle vous permet de gérer vos tâches quotidiennes à travers plusieurs listes personnalisables, le tout dans une interface moderne et réactive.

macOS 14.6+ SwiftUI Français / Anglais

Fonctionnalités

  • Multi-Listes : Organisez vos tâches par contexte (Travail, Maison, Courses...).
  • Interface Post-it : Code couleur intuitif (Jaune pour à faire, Orange pour fait).
  • Mode Sombre : Support complet du mode sombre système.
  • Horloge Analogique : Une touche classique pour garder le temps en vue.
  • Import / Export : Vos données vous appartiennent (TXT, CSV, Markdown).
  • Tip Jar : Possibilité de soutenir le développeur via l'application.

Guide Rapide

Créer une liste

Cliquez sur le bouton + dans la barre latérale ou utilisez le menu contextuel pour ajouter une nouvelle catégorie de tâches.

Ajouter des tâches

Utilisez le bouton + principal. Astuce : Vous pouvez ajouter plusieurs tâches à la fois en les séparant par un point-virgule (ex: Pain; Lait; Beurre).

Nettoyage

Le bouton ∆ permet de supprimer rapidement toutes les tâches terminées de la liste en cours.
