# Captures d'écran

Déposer ici les fichiers :

- `01.png` — vue principale (rose des Nords + matrice).
- `02.png` — détail d'une branche (calcul vent, GS, temps).

Formats acceptés pour App Store Connect :
- iPhone 6.9" : 1320 × 2868 px
- iPhone 6.5" : 1242 × 2688 px
- iPad 13"   : 2064 × 2752 px

Les images sont référencées depuis `index.html`. Si elles sont absentes, un placeholder s'affiche automatiquement.
